# Great Lakes Investment Fund — Website

A professional financial institution website for the **Great Lakes Investment Fund**, a private capital fund offering accredited investors access to income-generating opportunities through structured bond offerings. Managed by Seifert Capital Partners Inc., Troy, Michigan.

---

## Design System — "Meridian Wealth"

| Token | Value |
|---|---|
| Primary Color | Obsidian Navy `#0A1628` |
| Accent Color | Burnished Gold `#C8A96E` |
| Heading Font | Playfair Display (serif) |
| Body Font | Outfit (sans-serif) |
| Design Philosophy | Contemporary Luxury Finance |

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | React 19 |
| Styling | Tailwind CSS 4 |
| Language | TypeScript |
| Routing | Wouter |
| Animations | Framer Motion + CSS |
| Icons | Lucide React |
| Build Tool | Vite 7 |
| Package Manager | pnpm |

---

## Pages

| Route | Description |
|---|---|
| `/` | Home — Detroit skyline hero, stats, investment approach, FAQ |
| `/offerings` | Bond offerings (12/17/25/43 month terms), investment process |
| `/about` | Company story, Mission/Vision/Philosophy, leadership |
| `/contact` | Contact form, office details (Troy, MI) |
| `/faqs` | 15 detailed Q&As across 5 categories |

---

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

---

## Key Features

- **Detroit Renaissance Center** skyline hero image (AI-generated, dusk)
- Animated stat counters (51 projects, $8M+ deployed, 13.5% APY)
- 4 bond term investment cards with APY rates
- 6-step investor onboarding process
- Webinar registration forms
- Full FAQ accordion with category navigation
- SEC Regulation D, Rule 506(c) compliance notices
- Fully responsive — mobile, tablet, desktop
- Scroll-triggered fade-in animations
